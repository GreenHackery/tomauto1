## About this node

#### To run this script outside of ros
1. cd into `greenhackery_camera/greenhackery_camera`
2. run `./ImagePublisher.py`

#### To run this as a standalone ros2 node
1. cd to BASELINE_4wd
2. run `colcon build`
3. run `source ~/.bashrc`
4. run `ros2 run greenhackery_camera image_publisher`
